论文管理

在学术的过程中，论文的管理是一个非常头疼的事情，当需要整理的论文多到一定数量的时候，很难按照一种有秩序的方式来管理。但是科技作为第一生产力，总是有办法来解决这个问题的。 Paper3 无疑是提升生产力的一个优秀的代表。阅读论文很难说是一个很好的体验，阅读的难度就不说了，还需要从中提炼出我们所需要的信息。如果不做好这些笔记，那么过了一会(并不需要一天)，这篇论文里面讲的东西也就忘的差不多了。在学术的过程中，有些问题总是一直存在的:

  • 论文的分类
  • 获取高质量的论文
  • 获取一篇特定的论文
  • 写论文想要用之前的笔记或者批注，发现很难找到
  • 论文丢了。。或者电脑丢了。。。。

而 Paper3 刚好就可以用来解决这些问题。

强大的搜索能力

在 Paper3 里面，内置了多达 25 中搜索引擎，常见的例如 ACM, Google Patents, Google Scholar, Web of Science 等数据库引擎，所以如果想要找的论文在这里找不到，在其他的地方也不大可能找到了。

在搜索的过程中，所输入的关键词会同时在你收藏的数据库中搜索。而且同时支持从多方面的搜索，比如 title, author, year, publisher, abstract 方面搜索，比如我要搜索一篇 SDN 在 2015 年发表的论文，我可以如下搜索:

整洁的资料分类

Paper3 提供了强大的论文分类功能，可以很方便的建立 Collection,我们可以通过 File -> New Collection 来新建一个 Collection。Collection 总体上可以分成两类:

  • 一类是普通的 Collection, 这样我们就需要通过右键点击论文 add to collection

  • 还有一个是 Smart Collection,论文会根据你自定义的参数自动的加入进来。

除了这样的分类，还可以为文章添加关键词，随后在搜索的时候，我们可以直接搜索这些关键词，就可以快速的定位到我们看过的一些文章，就可以自然的找到做过的笔记和批注。

便利的阅读体验

以下是阅读的主界面:

右栏中可以显示论文的相关信息，和笔记以及一些社交信息。

在阅读论文的过程中，可以通过右键来完成以下的功能:

  • 可以轻松的在论文里面标注，加自己的笔记
  • 将一段文本标记为高亮，有多种颜色可选
  • 将一段文本标记为下划线
  • 将一段文本标记为删除线
  • Mac 利用自带的词典可以随时查单词，在 Windows 上可以利用有道词典的屏幕划词功能来取代这个。
