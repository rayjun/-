HBase 常用 shell 命令手册

HBase 是一个开源的分布式非关系型数据库，在大数据行业中应用很广泛。Google 的三篇论文奠定了大数据的理论基础，其中 HBase 就可以看做是 BigTable 的实现，用来存储大规模的数据。

HBase 的命令很多，这篇文章总结了 HBase 最常用的 shell 命令，作为日常开发的手册。

本文基于 JDK1.8 HBase2.2.5 本文假设你已经了解 HBase 的基本概念
